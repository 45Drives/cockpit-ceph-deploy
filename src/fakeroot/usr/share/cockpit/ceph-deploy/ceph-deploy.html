Pre-configuration Requirements

Please ensure that the following network steps are completed before proceeding:

  • Each host should be able to ping all other hosts using a hostname.
  • Any bonds that are required have already been configured and tested.
  • ssh keys have been generated on the administrator node and passwordless ssh to all hosts is working.

Once these requirements are met, click "Done" to complete this step and return to the main menu.
